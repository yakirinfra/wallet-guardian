\# Wallet GuardianA lightweight custody policy engine that simulates institutional wallet management flows for BILS stablecoin.\## Demo<p align="center">&#x20; <img src="assets/demo.gif" alt="Wallet Guardian Demo" width="900"></p>\---\## OverviewThis project demonstrates how financial institutions manage digital assets using a \*\*control layer before execution\*\*.Instead of sending funds directly, each transaction is evaluated against predefined security policies.\---\## Demo Flow (What is shown in the video)\### 1.
